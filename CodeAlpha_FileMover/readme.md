JPG FILE MOVER 
This simple python program helps the user to move photos from one folder to another one.
Key Features :
1)Creates the destination folder by itself if a folder doesnt  exist in the name you entered in destination folder.
2)Handles errors such as no photos found , no source file found , file permission issues gracefully .
3)Has animateed spinner and real time feedback during file transfer.
4)Has user-engaging text animations and absolute file path for maxmimum file transperacy.
How it works :
1)Change working directory to the script's location.
2)Validates source folder existence.
3)Creates destination folder if needed.
4)Scans source folder for .jpg files (case-insensitive).
5)Moves each JPG file with visual progress indicator.
6)Displays summary of moved files.
If you are a person who wants to keep their photos organized then this program might be useful to you.
